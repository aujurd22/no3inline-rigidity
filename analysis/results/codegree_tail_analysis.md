## 对吸收法路线（dir #2）的修正建议
不要"删除危险配对"，改为 **分级处置**：
1. **硬禁止** 666 个 co3≥800 极端配对（匹配结构，零代价）；
2. **软惩罚** 143k 个中等配对（200≤co3≤400）通过加权目标 / lopsided LLL / nibble
   下权，而非硬删；
3. absorber 在超边层操作：先构造近冲突自由 2-因子，再用预留 cell 补齐度数。
4. 这等价于把 dir #2（异常对删除+稀疏化+吸收）与 dir #5（加权 switch 势）合并：
   势函数 `Φ(F)=B(F)+λ·Σ_{co3(u,v)>τ} w(co3(u,v))·1[u,v⊂F]`。

几何上危险配对集中在转置对+gcd≥2 方向，说明可在构造 2-因子时显式规避
"同时选 (i,j) 与 (j,i)"这一小类，作为高效的对称性剪枝。
